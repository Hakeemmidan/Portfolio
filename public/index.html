lsdir - A Shell Alias for Only Listing Directories

The Problem When working in the terminal, you sometimes need to list only the directories in the current location. The standard ls command shows both files and directories, which can be overwhelming in cluttered directories. The command for listing only directories in the current location is ls -d */ which is not simple to remember, especially if it’s not something that you do often. The Solution: lsdir lsdir, an alias for only listing directories, which hopefully is easier to remember.
